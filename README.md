#GIW
En este repositorio recopilo todas las prácticas realizadas para la asignatura Gestión de la Información en la Web (GIW) del Grado en Ingeniería Informática de la Universidad Complutense de Madrid.
El objetivo de la asignatura es aprender a obtener, procesar y gestionar información procedente de la web, así como a desarrollar aplicaciones y servicios web seguros utilizando Python. Aquí se incluyen los distintos scripts y proyectos correspondientes a cada práctica.
A lo largo del curso he trabajado con diversas herramientas y tecnologías, entre ellas: Python 3, bases de datos como SQLite, MySQL y MongoDB, manejo de datos en XML, CSV y JSON, y frameworks web como Flask y FastAPI. También he aplicado técnicas de web scraping, consumo de APIs y conceptos básicos de seguridad web.

In this repository, I compile all the assignments completed for the course Web Information Management (GIW), part of the Computer Engineering Degree at the Complutense University of Madrid.
The aim of the course is to learn how to retrieve, process, and manage information from the web, as well as how to develop secure web applications and services using Python. This repository includes the different scripts and projects corresponding to each assignment.
Throughout the course, I have worked with various tools and technologies, including Python 3, databases such as SQLite, MySQL, and MongoDB, data handling with XML, CSV, and JSON, and web frameworks like Flask and FastAPI. I have also applied techniques such as web scraping, API consumption, and basic concepts of web security.
